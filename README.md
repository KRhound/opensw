# 명령어 조사
 ## 1. top
 ```-a : 메모리 사용에 따라 정렬
-b : 배치 모드에서 시작
-c : 명령어 대신 명령어 라인을 보여줌
-d : 업데이트 간격을 조정
-h : 도움말
-H : 모든 개별 쓰레드가 보여짐
-i : 좀비(zombie) 또는 Idle 상태의 것들은 무시됨
-m : VIRT 대신 USED를 보고
-M : 메모리 유닛(K/M/G)을 보여줌
-n : 반복의 최대 수를 지정
-P : 지정된 프로세스 ID들만 보여줌
-s : 보안 모드로 시작
-S : 누적 시간 모드로 시작. 활성화되면 각 프로세스는 CPU를 사용한 시간과 함께 출력
-u : 지정된 유효 사용자에 의한 프로세스만 보여줌
-U : 지정된 사용자에 의한 프로세스만 보여줌. 사용자는 실제, 유효한, 저장된 및 파일시스템 UID를 의미
-v : 프로그램 라이브러리 버전을 출력```
